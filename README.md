# LMS Frontend

## Setup instruction

1. Clone the project

```

    https://github.com/pankajsharma81/LMS.git
```

2. Move into the directory

```
    cd lms-frontend-hn
```

3. install dependencies

```
    npm i
```

4. run the server

```
    npm run dev
```
